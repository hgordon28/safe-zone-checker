# safe-zone-checker

Small, dependency-free creator-marketing tools. Each tool is a single HTML file — open it in a browser, no build or install needed.

## Tools

- **`index.html` — Ad safe zone checker.** Drop a vertical photo/video and preview platform-safe zones (Meta Reels/Stories, TikTok, YouTube Shorts, Snapchat) plus the combined strictest safe area.
- **`campaign-tracker.html` — Influencer campaign tracker.** Start-to-finish tracker for influencer marketing campaigns:
  - **Aggregate dashboard** across all campaigns: revenue, spend, profit and blended margin, deliverables posted, overdue items, plus a per-campaign table and a unified "needs attention / next 14 days" deadline feed.
  - **Budget & profit** per campaign: revenue, cost budget, line-item costs with categories (creator fees, shipping, travel & flights, misc…), budget burn bar, profit and margin computed automatically.
  - **Delivery targets**: views, engagements, posts live, etc., with attainment bars.
  - **Deliverables** with status pipeline, due dates, live links, and **separate Brand / Creator / Internal comment threads** per deliverable.
  - **Deadlines**: per-campaign timeline combining milestones and deliverable due dates, with quick-add milestone presets and overdue/upcoming badges.
  - **Custom details & notes**: free-form label/value fields (product details, shipping addresses, discount codes, usage rights…) and an autosaving notes pad.
  - Data is stored in your browser (localStorage), with JSON export/import for backup or moving between machines. A "Load sample data" button shows a fully populated example.
